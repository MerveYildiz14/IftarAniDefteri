🔧 Firebase Kurulum Rehberi

Bu dosyayı Netlify'a atmadan önce sadece Firebase bilgilerini girmen gerekiyor. 5 dakika sürer.

Adım 1
Firebase'e kaydol

firebase.google.com → Google hesabınla giriş yap → Add project → projeye bir isim ver (örn: "ani-defteri") → devam et

Adım 2
Firestore veritabanı oluştur

Sol menü → Firestore Database → Create database → Start in test mode → Konum seç (europe-west1 önerilir) → Enable

Adım 3
Web uygulaması ekle & config al

Sol üst → Proje ayarları (dişli ikon) → Your apps → </> ikonuna tıkla → kayıt ol → çıkan firebaseConfig nesnesini kopyala

Adım 4
Bu dosyayı düzenle

Dosyayı bir metin editörüyle aç (Not Defteri de olur). En üstteki FIREBASE_CONFIG bölümüne kopyaladığın bilgileri yapıştır:

apiKey: "AIzaSy...", authDomain: "ani-defteri.firebaseapp.com", projectId: "ani-defteri", ...
Adım 5
Netlify'a yükle

netlify.com/drop → HTML dosyasını sürükle bırak → otomatik link al → QR koda çevir!

Not
Test modunu aktifleştir (önemli!)

Firebase Console → Firestore → Rules → şu kuralı yapıştır ve yayınla:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }

Bu rehberi tekrar görmek için sayfanın altındaki "Kurulum Rehberi" linkine tıkla.
